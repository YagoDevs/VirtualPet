═══════════════════════════════════════════════════════════════════
🎨 DICAS PARA ESCOLHER CORES:
═══════════════════════════════════════════════════════════════════

✓ Contraste é importante!
  → Use cores bem diferentes para 100% e 0%
  → Facilita visualizar rapidamente o status

✓ Pense no significado:
  → Verde = saudável, cheio
  → Vermelho = perigo, vazio
  → Amarelo = atenção, médio
  → Azul = energia, frescor

✓ Combine com o tema do seu pet:
  → Pet fofo = cores pastéis/vibrantes
  → Pet sério = cores sóbrias
  → Pet mágico = roxo, ciano, magenta

✓ Teste no jogo:
  → As cores ficam diferentes na tela
  → Teste em diferentes momentos (dia/noite se tiver)

✓ Use cores complementares:
  → Verde ↔ Vermelho
  → Azul ↔ Laranja
  → Roxo ↔ Amarelo
